Intro

My name is Vamshi Renduchintala, and I am a full stack software engineer graduating from App Academy. I previously worked as a scientist at a molecular diagnostics company called Cepheid, before deciding to pursue my true interest and make the switch to software development. My primary experience so far is in developing fully functioning web applications utilizing React and Redux frontend, along with Ruby on Rails backend / Node.js backend. In my free time, I love fitness, travel, spending time outdoors, concerts, and whatever latest hobby I've picked up!

Resume

Willow

Within a 12 day timeframe, I built a full stack clone of the real estate web application Zillow, humorously titled Willow. The backend was built using Ruby on Rails, with a PostgreSQL database. Local and global state shaping was carefully crafted utilizing jBuilder and the Redux library, and rendering the components of the website using a React frontend. As my first experience building a fully functional app which uses a RESTful API, I was able to cement my knowledge and understanding of fundamental web architecture throughout this process.


Smooth

I took a weekend to learn the MERN stack, and worked with a team of 4 other software engineers to build a day trip planner app titled Smooth. MongoDB was utilized as our database, using Amazon Web Services (AWS) for image hosting. Our backend was built using Node.js and Express.js, with a React frontend. Redux library was used to structure the global state of the application, React DnD API to enable drag and drop features, and Chart.js for data visualization. This experience really honed my adaptability, as I quickly learned a new tech stack and implemented those same fundamental web design principles I had come to learn, along with adding new tools and libraries to my repertoire.


Deep Space Destroyers

I used Vanilla JavaScript and Canvas API along with HTML and CSS styling to create an interactive single player game in 4 days, titled Deep Space Destroyers. This taught me my understanding of game development concepts such as game loops, player input handling, collision detection, and scoring systems, and proved to be an enjoyable way to cement my passion for programming.
